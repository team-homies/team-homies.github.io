# HOHOBOOK (도서 관리 프로그램)
## 1. 프로젝트
  - 요구사항
    - 도서 리스트 조회
    - 도서 제목 조회
    - 도서 상세정보 조회
    - 대여자 조회
    - 도서 정보 수정
    - 도서 등록/삭제
    - 도서 대출/반납
  - 기술스택
    - golang : 기능 구현 시 사용되는 언어
    - mysql : 데이터 등록 시 사용되는 DB
    - gorm : DB연동 시 사용되는 ORM
    - gin-gonic : API구현 시 사용되는 프레임워크
  - 환경세팅
    - vscode
    - docker
    - dbeaver
  - 일정
    - 1/30(화) - 1/31(수) : 설계, 작업 배분 끝내기 (2일)
    - 2/1(목) - 2/6(화) : 프로토타입 완성 (6일)
    - 2/7(수) - 2/10(토) : 디버깅 및 보완 (4일)
    - 2/11(일) - 2/12(월) - 테스트 기간 (2일)
